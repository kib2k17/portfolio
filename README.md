## Deploy on GitHub Pages

1. Create a new repository and push this folder’s contents (or make this folder the repo root).
2. In the repository on GitHub: **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Choose branch **`main`** (or `master`) and folder **`/ (root)`**, then save.

The site will be available at `https://<username>.github.io/<repo>/` (or at the root domain if you use a `username.github.io` repository).

To use a custom domain, add a `CNAME` file at the repo root with your domain name and configure DNS per GitHub’s docs.
